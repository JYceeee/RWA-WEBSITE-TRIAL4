Trade Project 页面功能特性

  • 项目信息展示 - 显示完整的项目详情和指标
  • 交易类型选择 - 支持买入(Buy)和卖出(Sell)操作
  • 数量输入 - 用户可输入交易数量
  • 价格设置 - 支持市价单和限价单
  • 交易总结 - 实时计算交易总金额
  • 交易历史 - 显示最近的交易记录
  • 响应式设计 - 适配移动端和桌面端
  • 面包屑导航 - 方便用户返回上一页

使用说明

  1. 访问Projects页面 - 在Vue应用中访问 /projects 页面
  2. 点击Trade按钮 - 在任意项目卡片上点击橙色的Trade按钮
  3. 进入交易页面 - 自动跳转到 /trade/{项目代码} 页面
  4. 选择交易类型 - 选择买入(Buy)或卖出(Sell)
  5. 输入交易信息 - 填写交易数量和价格设置
  6. 提交交易 - 确认交易信息后提交订单

技术实现细节

  • 路由参数传递 - 通过URL参数传递项目代码
  • Props接收 - 组件通过props接收项目代码参数
  • Session存储 - 项目信息存储在sessionStorage中
  • 响应式数据 - 使用Vue的响应式系统管理状态
  • 计算属性 - 实时计算交易总额
  • 事件处理 - 处理用户交互和表单提交
  • CSS Grid布局 - 使用现代CSS布局技术
  • 移动端适配 - 响应式设计确保移动端体验
